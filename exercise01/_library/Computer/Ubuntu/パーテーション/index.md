パーテーションとは  
パーテーションの方式は、Windwos, Linuxともに以前はmaster boot record(MBR)という方式が使われていたが、現在では、GUID Parition Table(GPT)という方式が使われている。

一般的に、それぞれのディスクデバイスは、一つのパーテーションテーブルを持つが、Mac OS Xは、このルールに反する。
